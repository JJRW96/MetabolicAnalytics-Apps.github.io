4. Key Efforts Analysis
ⓘ This identifies key 'breakthrough' efforts relative to the currently fitted model curve and displays them as star markers on the plot.

Selection Logic:
- Always includes the max power for 1s and 20s.
- For several time buckets (e.g., 1-3m, 5-10m, etc.), it finds the point with the highest positive percentage deviation from the model and adds it if it's above the curve.

Special Rule:
- For the critical fitting time-ranges, if no point is above the curve, it will instead select the point with the smallest negative deviation to provide a reference.
